# Student Performance Prediction System

## Developed by
1. Mobeen Maroof

## Project Description
The Student Performance Prediction System is a Flask-based Machine Learning web application that predicts whether a student is likely to pass or fail based on academic factors such as study hours, attendance, previous grades, and other relevant information. The system helps educators identify at-risk students and make data-driven decisions to improve academic performance.

## Features
- Predict student performance (Pass/Fail)
- User-friendly Flask web interface
- Real-time prediction results
- Machine Learning model integration
- Student data input and analysis
- Accurate and efficient performance prediction

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS
